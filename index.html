Video links

The source could be stored in a folder that is shared with the html file. Look up the different syntax for linking local files (blank, .. or /)

<video width="320" height="240" controls autoplay>
<source src="Video/video.mp4" type="video/mp4">
Your browser does not support the video tag. </video>

Audio links

<audio controls autoplay loop> <source src="Audio/audio.mp3" type="audio/mpeg"> Your browser does not support the audio element. </audio>

Embedding Video from Youtube

  1. Visit a youtube video.
  2. Select SHARE
  3. Select EMBED
  4. You can select all or some of the embed code. Specifically the unique movie code which tends to be a series of numbers/letters.
COMPLEX EMBED

<iframe width="560" height="315" src="h ttps://www.youtube.com/embed/2QyvO5ainpw" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
